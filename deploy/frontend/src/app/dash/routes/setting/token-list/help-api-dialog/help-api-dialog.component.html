How to use your API Key

Use this snippet to test your API Key in client like Postman, the terminal or whatever you want. 

curl -X 'POST' \
'https://streams.krules.io/api/v1/0/to-remove/001' \
-H 'accept: application/json' \
-H 'api-key: [YOUR_API_KEY]' \
-H 'Content-Type: application/json' \
-d '{"temp":25}'
